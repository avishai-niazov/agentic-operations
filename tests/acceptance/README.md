# Acceptance Checks

Use this folder for outcome-level checks that can be verified without relying on persuasive prose.

Suggested files:
- `content_publishability_checks.md`
- `seo_safety_checks.md`
- `technical_change_checks.md`
- `memory_promotion_checks.md`

Each acceptance artifact should state:
- scope
- observable checks
- pass/fail result
- unresolved risks
