# Review Checklist — Content Production

This is a **pre-route self-check**. Walk it before sending a draft to guardian review. If any item fails, fix it before routing.

## 1. Brief alignment

- [ ] The draft addresses the approved brief's audience, learning goal, target surface, teacher intent, and no-go zones.
- [ ] Scope drift (new topics, expanded audience, changed goal) did not occur.

## 2. Structure and shape

- [ ] The draft follows the required shape from `knowledge/content/content_templates_v01.md`.
- [ ] Headings reflect the actual structure of the content, not keywords alone.
- [ ] The draft has a clear entry point, a clear path through, and a clear takeaway.

## 3. Voice and distinctiveness

- [ ] The draft sounds like the established voice, not generic AI writing.
- [ ] No "in today's world," "as we all know," restated-question openings.
- [ ] Where possible, I compared a paragraph to an approved exemplar in `knowledge/content/exemplar_library_v01.md`.

## 4. Pedagogy surface

- [ ] The learning goal is visible to the reader, not only to the author.
- [ ] Teacher usefulness is preserved (teacher-use block present where the template requires one).
- [ ] No shallow busywork; activities and examples do real pedagogical work.

## 5. Claims and grounding

- [ ] Every factual claim is supported or could be supported on request.
- [ ] No invented sources, statistics, or citations.
- [ ] No overreach beyond the audience's context.

## 6. Metadata and supporting blocks

- [ ] Title, meta description, and social metadata drafts are present and distinctive.
- [ ] FAQ section (if part of the template) answers real audience questions, not filler.
- [ ] Internal-link suggestions point to real, relevant pages, not random hub padding.

## 7. Non-negotiables

- [ ] No violation of `knowledge/content/non_negotiables_v01.md`.
- [ ] No competitor-style phrasing imported from the market-intelligence corpus.
- [ ] No market-intelligence content used as voice source.

## 8. Routing readiness

- [ ] Draft is saved to `queues/awaiting_guardian_review/`.
- [ ] Brand review and pedagogy review are both requested.
- [ ] A trace record is emitted.

## 9. Prohibited-action check

- [ ] No attempt to publish directly.
- [ ] No rewrite of constitutional files.
- [ ] Not optimized for keywords at the expense of usefulness.
