# LinkedIn Summary v02

I built a harness-driven agentic operations framework for Planerium — not as a generic “AI content engine,” but as a governed operating model for a real educational website.

The key idea is simple:
AI should help scale operations without erasing what makes a product distinctive.

So the system uses:
- narrow specialist agents
- typed handoffs
- a harness orchestration layer
- self-learning through traces and evaluation
- self-healing through rollback, quarantine, and escalation
- constitutional safeguards for brand distinctiveness and pedagogical integrity

That last part matters most. In education, scale without identity and quality is not a moat. It is drift.
