# Regression Tests v02

## Purpose

Preserve lessons from past failures.

## Example regression cases

- content sounds generic after SEO optimization
- educational explanation removed to improve brevity
- ad test harms printable usability
- template change breaks series navigation
- crawl fix introduces wrong canonical pattern

## Test shape

- regression_id
- origin
- failure summary
- trigger condition
- expected safe behavior
- fail indicator
- severity
