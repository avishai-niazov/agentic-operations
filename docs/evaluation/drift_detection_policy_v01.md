# Drift Detection Policy v01

## Purpose

Detect slow degradation in voice, identity, and educational quality.

## Drift classes

- brand drift
- pedagogy drift
- market imitation drift
- quality threshold drift
- template-filler drift

## Detection signals

- repeated guardian rejections
- repeated conditional passes with same weakness
- rising filler language
- reduced teacher-usefulness score
- output shape becoming more competitor-like

## Response

1. open drift case
2. quarantine weak pattern
3. tighten review threshold if justified
4. add regression test
5. route upstream for structural correction
