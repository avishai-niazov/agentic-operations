# Guardian Gate Model v01

## Purpose

Define the four required guardian gates for content-related work.

## Gate 1 — Brief Brand Validation

Owner: Brand Steward

Question: Does the idea fit site identity before production begins?

Checks:
- unmistakably site-specific or generic
- enrichment-first value
- tone fit
- originality fit
- avoidance of SEO filler

## Gate 2 — Brief Pedagogy Validation

Owner: Pedagogical Steward

Question: Is the idea educationally worthwhile before production begins?

Checks:
- learning goal clarity
- audience fit
- meaningful activity design
- teacher / parent usability
- differentiation value where needed

## Gate 3 — Draft Brand Validation

Owner: Brand Steward

Question: Did the produced output preserve site identity?

Checks:
- voice and clarity
- distinctiveness
- non-generic structure
- alignment with the site ecosystem
- trustworthiness

## Gate 4 — Draft Pedagogy Validation

Owner: Pedagogical Steward

Question: Did the produced output preserve educational quality?

Checks:
- age / grade fit
- instructional clarity
- teacher usefulness
- avoidance of shallow busywork
- physical-learning value where relevant

## Outcome classes

- `pass`
- `pass_with_conditions`
- `fail`

Any `fail` sends the work back to the previous operator stage.
