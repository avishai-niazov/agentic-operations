# Invariant Register v02

## Constitutional invariants

1. Brand distinctiveness must not be diluted for speed.
2. Pedagogical integrity must not be diluted for scale.
3. Market intelligence may guide opportunity, not voice.
4. Content cannot bypass guardian gates.
5. Risky writes cannot bypass governance.
6. Self-learning cannot rewrite constitutions or charters.
7. Self-healing must remain auditable.

## Operational invariants

8. Every run must emit a trace.
9. Every write-capable run must state its permission basis.
10. Every blocked result must record why.
11. Every promoted learning must have evidence.
12. Every rollback candidate must have explicit impact reasoning.
